# Audio jack mapping — macOS

🚧 **En cours de fabrication** — Work in progress.

Ce guide expliquera comment configurer un cabinet en 5.1 ou 7.1 sur macOS, via **Audio MIDI Setup** (Configuration audio et MIDI).

Particularité macOS : l'audio intégré des Mac ne supporte pas le retasking des jacks comme Linux/Windows. Pour un setup multi-canaux, il faut généralement passer par une interface USB externe ou un aggregate device.

À venir :
- Exemples d'interfaces USB multi-canaux compatibles (Behringer UCA222, Focusrite Scarlett, adaptateurs USB 5.1/7.1…)
- Création d'un **Aggregate Device** dans Audio MIDI Setup
- Capture d'écran avec la config cabinet recommandée
- Vérification côté PinReady / VPX
